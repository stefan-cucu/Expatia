## ğŸ Getting Started <a name = "getting_started"></a>
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

You require the following API keys:
* A Google API key inside the "src\pages\post.js" and "public\index.html" files, including the following APIs:
  * Maps JavaScript API
  * Places API
  * Geocoding API
* A Firebase API key inside the "Config.js" file in the root directory
